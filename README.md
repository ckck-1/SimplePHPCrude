# Simple PHP CRUD

This is a simple PHP CRUD (Create, Read, Update, Delete) application using SQLite. No frameworks or external dependencies are required.

## How to Use

1. Make sure you have PHP installed (with SQLite support).
2. Place the `index.php` file in your web server directory.
3. Open the app in your browser (e.g., `http://localhost/index.php`).
4. Add, edit, and delete items using the web interface.

The app will automatically create a `database.db` SQLite file in the same directory.
